Install

Click button. Add to Chrome. Pretend this was always healthy.

The Chrome Web Store is the polite path. The ZIP is for manual install people, brave souls who already know what Developer mode is and have made peace with their decisions.

  1. Polite path: open the Chrome Web Store listing and click Add to Chrome. Go outside.
  2. Manual path: download the ZIP. Unzip the ZIP. We are not making this up.
  3. Open chrome://extensions, flip Developer mode on, and click Load unpacked.
  4. Select the unzipped Doodler folder. It is annoying. It is legal. It is fine.
